# TUJUAN PROJECT

Project ini bertujuan membangun sebuah platform bernama FaceAI.

FaceAI bukan hanya aplikasi deteksi wajah.

FaceAI adalah sebuah Face Intelligence Engine yang nantinya dapat berkembang menjadi:

* Face Recognition
* Face Verification
* Face Analysis
* Face Landmark
* Face Shape Detection
* Face Attribute Analysis
* Skin Analysis
* Hair Analysis
* Face Recommendation
* Hairstyle Recommendation
* Makeup Recommendation
* Virtual Makeup
* Beauty Assistant
* Personal Stylist AI

Seluruh project harus dibangun secara modular sehingga semua modul dapat digunakan kembali pada project lain.
